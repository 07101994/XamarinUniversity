Lab goals

The goal of the exercise is to add the SQLite.NET component to each project in your solution using Nuget.

What is NuGet?

What is NuGet?
NuGet allows developers to easily download, install, update, and remove 3rd-party libraries within a Visual Studio project. These libraries are made available through the NuGet Gallery, which is an online repository managed by Microsoft. NuGet was created by Microsoft and is available in Visual Studio through the NuGet Package Manager which is built into Visual Studio.

If you did not complete the previous exercise, you can use the Lab.Completed solution from the previous part as starter code for this part.

Exercise challenge

Follow the above goals to complete the exercise. You can refer to the step-by-step instructions below to get more detailed information on each goal.

Steps

Add the SQLite.Net PCL Nuget package

Let's add the Nuget package which will allow us to use SQLite-Net. Make sure you add the correct package - the id is sqlite-net pcl, there are other packages with very similar names so check the entry carefully before you install it!

  1. If you are using Visual Studio for Mac, right-click the Packages folder in the People Portable Class Library project and select Add Packages... from the popup menu.
  2. In the Search field, enter sqlite-net pcl.
  3. Select SQLite-net PCL from the list of search results and then press the Add Package button in the lower right.
  4. If you are using Visual Studio on Windows, right-click on the People Portable Class Library project name within Solution Explorer, select Manage Nuget Packages....
  5. Select the Browse tab and enter sqlite-net pcl in the Search field.
  6. Select sqlite-net-pcl in the list of search results and press the Install button.
  7. Repeat the steps to add the SQLite.NET PCL nuget package for the other projects (iOS, Android, and Windows).

Summary

In this lab, you added SQLite.Net support to each project by adding the SQLite.Net PCL Nuget package. You're now ready to work with a SQLite database.
