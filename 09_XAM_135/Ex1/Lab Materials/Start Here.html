Overview

This courses explores the core ways you size and position the views in your UI. It begins by focusing on the size settings provided in the View class. It then covers the two most common layout containers: StackLayout and Grid. It finishes by showing how to use a ScrollView to scroll a layout.

Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Xamarin development environment with either a simulator/emulator or a physical device. Please see the setup page if you need guidance installing the Xamarin development environment.

Objectives

  1. Specify the size of a view
  2. Arrange views with StackLayout
  3. Apply Attached Properties
  4. Arrange views with Grid
  5. Scroll a layout with ScrollView
